# 组件开发指南

## 1. 初始化组件

在[创建依赖库](../20.快速开始/20.创建依赖库.md)之后, 执行 `lcap create` 命令可选择创建组件：

<img src="https://community.codewave.163.com/libraryUpload/app/U443ZAV8gRW99mZM6Ze6pirw7Pz65ueCKAjijgr2RvjAAKxUf232hD47TA18bev0H4S35x0evenB2YNQuywGsdwZqyOoAvCTsrD2a0KdV2k_20241122153757688_ori.jpg" class="imgStyle" style="" />

创建组件时，需要设置组件名称（组件使⽤大驼峰的格式命名，如CapsuleSelect），组件别名以及组件的适用终端，默认选择为PC端。如下：

<img src="https://community.codewave.163.com/libraryUpload/app/U443ZAV8gRW99mZM6Ze6pigsaffNDkorWN_VKUjVeMXAAKxUf232hD47TA18bev0H4S35x0evenB2YNQuywGsdwZqyOoAvCTsrD2a0KdV2k_20241122153746369_ori.jpg" class="imgStyle" style="" />

### 组件目录

<div class="highlight">

依赖库模板选择 Vue2 时（注：IDE版本>=3.8）

</div>


```
|- components
|-- cwd-casplue-switch                         # 组件名称
|---- stories
|------ block.stories.js                       # 运行示例，必需
|------ examples.stories.js                    # 调试demo
|---- api.ts                                   # 组件描述文件
|---- index.vue                                # 组件实现代码
|---- index.ts                                 # 组件导出文件
|---- index.module.css                         # 组件样式目录
```

***
<div class="highlight">

依赖库模板选择 React 时（注：IDE版本>=3.8）

</div>

```
|- components
|-- CasplueSwitch                               # 组件名称
|---- stories                                   # 调试demo 目录
|------ block.stories.tsx                       # 运行示例，必需
|------ examples.stories.tsx                    # 调试demo
|---- api.ts
|---- index.tsx                                 # 组件目录
|---- index.module.css                          # 组件样式目录
```

## 2. 开发说明

开发需要完成的内容如下图：

<img src="https://community.codewave.163.com/libraryUpload/app/U443ZAV8gRW99mZM6Ze6pgkdyni9g2B0hf5ibbTEJJnAAKxUf232hD47TA18bev0H4S35x0evenB2YNQuywGsdwZqyOoAvCTsrD2a0KdV2k_20241122153749183_ori.png" class="imgStyle" style="" />

- \[必要配置] [api.ts](../30.组件开发/15.组件配置编写说明.md)：组件配置说明， 描述用户组件的配置面板生成，应用翻译等。
- \[必要配置] [block.stories.{js|tsx|jsx}](../30.组件开发/40.区块示例说明.md)：组件拖转到画布后初始生成的代码示例。
- \[必要配置] [页面编辑器适配](../30.组件开发/30.页面设计器适配说明/10.页面设计器适配说明.md)：IDE 页面编辑画布适配，用于在页面上的展示。
- \[可选配置] IDE 功能适配 ：主题配置、国际化能力等扩展能力适配。
- \[可选配置] 平台能力适配 ：对平台提供能力的适配。
