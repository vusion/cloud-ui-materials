# 依赖库介绍

`cust_card_translate_lib` 是一个专注于提供卡片动态效果的组件库。其核心功能是实现具有特定平移或移动特效的卡片展示，旨在帮助开发者在应用中快速构建交互感强、视觉效果生动的卡片列表展示页面。

# 包含的组件和逻辑列表

- **组件**：`card-translate-view` (特效移动卡片)
- **逻辑**：无

# 功能说明

- **card-translate-view (特效移动卡片)**：该组件通过接收一个数据列表，将内容以带有特定特效的移动卡片形式渲染。它适用于需要展示多项信息且希望增加动态切换或平移交互效果的场景。

# API 说明

## card-translate-view

### 属性 (Attributes)

| 名称 | 类型 | 默认值 | 说明 |
| :--- | :--- | :--- | :--- |
| list | nasl.collection.List<any> | - | 列表内容，用于定义卡片显示的动态数据源 |

### 事件 (Events)

该组件暂无公开事件供外部监听。

### 方法 (Methods)

该组件暂无公开方法供外部调用。
