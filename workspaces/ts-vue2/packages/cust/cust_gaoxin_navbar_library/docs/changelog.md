# cust_gaoxin_navbar_library 更新日志

## 1.0.0

这是初始版本，包含以下主要功能：
- 提供核心组件 `gx-u-navbar-multi`（导航栏）、`gx-u-navbar-item-multi`（项）、`gx-u-navbar-group-multi`（分组）及 `gx-u-navbar-divider-multi`（分割线）。
- 支持通过数据源（DataSource）动态渲染导航项，并可自定义文本、值、图标及跳转链接字段。
- 集成路由联动功能，支持 Vue Router 的匹配与跳转配置。
- 提供丰富的交互属性，如链接打开方式（target）、弹出方向（placement）以及多种鼠标触发方式。
- 支持多级嵌套及折叠状态管理。

此版本基于当前代码状态生成，无历史变更记录。
