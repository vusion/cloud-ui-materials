# 选区依赖库 (cust_crop_view)

## 依赖库介绍

cust_crop_view 是一套专门用于实现选区功能的依赖库。它主要面向前端开发者，提供了便捷的图像或特定区域选择能力，适用于头像裁剪、图片局部处理及特定视觉区域定义等业务场景，帮助用户通过直观的交互方式完成区域锁定。

## 包含的组件和逻辑列表

- **crop-view (选区组件)**：用于加载图片资源并提供可视化选区操作的核心 UI 组件。
- **逻辑列表**：暂无自定义逻辑函数。

## 组件和逻辑功能说明

- **crop-view（选区组件）**：该组件能够渲染指定的图片源，并允许用户在其上方进行交互式的选区操作。支持监听图片加载状态，并提供了手动触发“完成”操作的方法，以便在业务流程中获取选区结果。

## API 说明

### crop-view（选区组件）

#### 属性 (Attributes)

| 名称 | 类型 | 默认值 | 说明 |
| :--- | :--- | :--- | :--- |
| 图片 (img) | nasl.core.String | '' | 需要进行选区操作的图片地址 |

#### 事件 (Events)

| 名称 | 说明 |
| :--- | :--- |
| 图片加载完成 (onImgLoad) | 当图片资源成功加载至组件内部时触发 |
| 完成时 (onFinish) | 选区操作确认或完成时触发，用于接收选区数据 |

#### 方法 (Methods)

| 名称 | 说明 |
| :--- | :--- |
| 完成 (finish) | 调用该方法可手动结束选区状态并触发完成回调 |

### 逻辑列表

暂无：该依赖库目前未包含独立的逻辑处理函数。
