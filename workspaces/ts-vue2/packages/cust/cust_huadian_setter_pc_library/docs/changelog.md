# cust_huadian_setter_pc_library 更新日志

## 1.0.0

这是初始版本，包含以下主要变更：

- 提供了两个核心扩展组件：`ex-i-ico`（图标-扩展）和 `ex-u-image`（图片展示-扩展）。
- 集成了自定义设置器：`ExIconSetter` 用于图标选择，`ExImageSetter` 用于图片资源配置。
- `ex-i-ico` 增强了图标布局、链接跳转类型及目标窗口的配置能力。
- `ex-u-image` 增强了对齐方式、填充策略、圆形遮罩以及加载过程中的占位图控制。
- 建立了基础的事件响应体系，包括鼠标交互、加载状态等。
- 该版本基于当前代码状态生成，无历史变更记录。
