# 中广核模型渲染依赖库 (cust_cgn_model_render_library)

中广核模型渲染依赖库是一个专注于核电业务领域的模型渲染组件库。该库旨在为开发者提供标准化的 UI 组件，用于展示反应堆热力性能指标等关键业务数据，提高核电监控与分析类应用的开发效率。

## 包含组件

- **反应堆热力性能主控界面 (reactor-thermal-performance-interface)**：用于展示反应堆热力性能相关的核心数据指标与主控界面。

暂无逻辑内容。
