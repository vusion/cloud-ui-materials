# cw_global_loading

## 介绍
cw_global_loading 是一款专为全局加载状态设计的依赖库。其主要功能是为应用程序提供统一的 Loading 效果管理方案，确保在执行异步任务、接口请求或页面切换时，能够为用户提供清晰的视觉反馈，提升产品的交互体验和专业感。
