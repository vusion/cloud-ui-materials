# cw_pdf_preview (pdf预览组件（基于浏览器原生）)

## 依赖库介绍
cw_pdf_preview 是一款基于浏览器原生渲染能力的 PDF 预览组件库。它旨在提供一种轻量且高效的方式，在 Web 应用中直接展示 PDF 文档，利用浏览器内置的 PDF 查看器确保兼容性与性能。

## 包含组件
- pdf-preview（PDF预览组件）

## 组件功能说明
### pdf-preview
用于在页面中嵌入并展示 PDF 文件。通过简单的链接配置，即可调用浏览器原生的 PDF 渲染引擎进行文档预览，支持基本的缩放、翻页及打印等原生功能。

## API 说明

### pdf-preview

#### 属性 (Attributes)
| 名称 | 类型 | 默认值 | 说明 |
| :--- | :--- | :--- | :--- |
| url | nasl.core.String | '' | 预览链接，指向需要展示的 PDF 文件地址 |

#### 事件 (Events)
暂无。该组件目前未定义任何自定义事件。

#### 方法 (Methods)
暂无。该组件目前未提供可供外部调用的实例方法。
