# 三维模型库 (cw_three_dimensional_model_library)

## 依赖库介绍

`cw_three_dimensional_model_library` 是一款专门用于集成三维模型展示能力的依赖库。它提供了标准化的组件，帮助开发者在应用中快速实现三维模型的加载、渲染与预览功能，适用于产品展示、三维可视化等业务场景。

## 包含的组件和逻辑列表

### 组件
- **model-preview**：三维模型预览组件。

### 逻辑
- 本库暂未包含全局逻辑。

## 组件功能说明

### model-preview (三维模型预览)
用于在页面中渲染三维模型。开发者仅需提供模型的链接地址，即可在界面上快速呈现模型的三维形态，支持基础的显示与集成。

## API 说明

### model-preview (三维模型预览)

#### 属性 (Attributes)

| 名称 | 标题 | 类型 | 默认值 | 说明 |
| :--- | :--- | :--- | :--- | :--- |
| link | 地址 | String | '' | 模型的链接地址 |

#### 事件 (Events)

暂无自定义事件。该组件目前仅用于展示，暂未定义交互相关的回调事件。

#### 方法 (Methods)

暂无公开方法。该组件目前的逻辑完全由属性驱动，暂未提供外部调用的实例方法。
