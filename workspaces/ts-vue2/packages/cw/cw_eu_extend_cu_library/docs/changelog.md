# cw_eu_extend_cu_library 更新日志

## 1.0.0
- 这是初始版本，包含以下核心组件：
    - `u-table-view`（数据表格）：支持复杂的数据展示、分页、排序、过滤及树形结构。
    - `u-table-view-column`：提供多种列类型（序号、选择、编辑等）及固定列功能。
    - `u-table-view-column-dynamic`：支持基于数据源动态渲染表格列。
    - `u-table-view-column-config`：支持用户自定义列的展示。 
    - `u-table-view-column-group`：支持表头分组。
    - `u-table-view-expander`：提供可自定义的节点展开/收起图标。
- 基于当前代码状态生成，无历史变更记录。
