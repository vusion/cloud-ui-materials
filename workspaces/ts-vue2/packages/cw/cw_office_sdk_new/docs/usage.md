# cw_office_sdk_new (新版文档SDK依赖库)

## 依赖库介绍
cw_office_sdk_new 是新版文档SDK依赖库，主要用于在应用中集成增强的文档处理与选择功能，支持 PDF 文件的预览、选择、访问策略控制以及修改后的文件上传等业务场景。

## 包含的组件与逻辑

### 组件列表
- **cw-pdf-select-view (pdf选择容器)**：提供 PDF 文件的展示与选择功能，支持配置访问权限，并允许导出或上传处理后的文件。

### 逻辑列表
- 暂无

## API 说明

### cw-pdf-select-view (pdf选择容器)

#### 属性 (Attributes)
| 名称 | 标题 | 类型 | 默认值 | 说明 |
| :--- | :--- | :--- | :--- | :--- |
| value | 文件地址 | string | - | 需要传入的 PDF 文件 URL 地址 |
| access | 文件访问策略 | string | public | 设置文件的访问权限。可选值：public (任何人可访问)、private (用户登录后可访问) |

#### 事件 (Events)
- 暂无：该组件目前未定义触发事件。

#### 方法 (Methods)
| 名称 | 标题 | 参数 | 说明 |
| :--- | :--- | :--- | :--- |
| uploadFile | 上传修改后的pdf | - | 调用此方法将修改后的 PDF 文件上传至服务器 |
| getSelected | 获取选择的列表 | - | 获取当前组件中已选择的文件列表数据 |
