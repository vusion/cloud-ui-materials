# 泵房三维模型 (cw_pump_house_model_library)

`cw_pump_house_model_library` 是一个专注于泵房三维模型展示的扩展库，旨在为用户提供直观、标准化的泵房 3D 场景组件，方便在应用中集成泵房相关的视觉呈现功能。

## 包含组件
- **pump-house (泵房)**：提供泵房的三维模型展示。
