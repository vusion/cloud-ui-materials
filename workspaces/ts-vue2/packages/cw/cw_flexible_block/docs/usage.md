# cw_flexible_block 两栏伸缩布局

## 依赖库介绍
cw_flexible_block 是一个用于实现两栏伸缩布局的组件库。它提供了左右分布的容器结构，并允许用户通过交互动态调整两侧的宽度比例，旨在为应用提供更具弹性的界面布局能力，非常适合用于侧边导航、详情预览或多面板管理界面。

## 包含组件与逻辑列表
### 组件
- resizable-columns（伸缩布局）：实现两栏宽度可调节的核心容器组件。

### 逻辑
- 暂无

## 组件功能说明
### resizable-columns（伸缩布局）
该组件将布局划分为左侧和右侧两个区域，支持配置左侧区域的初始默认宽度。用户可以根据需要向左右插槽中放入不同的业务组件，从而构建出一个可自由伸缩的页面结构。

## API 说明
### resizable-columns（伸缩布局）
#### 属性 (Props)
| 属性名 | 类型 | 默认值 | 说明 |
| :--- | :--- | :--- | :--- |
| leftWidth | nasl.core.Integer | 200 | 默认左侧容器的初始宽度 |

#### 插槽 (Slots)
| 插槽名 | 说明 |
| :--- | :--- |
| slotLeft | 默认左侧容器插槽，用于放置布局左侧的内容 |
| slotRight | 默认右侧容器插槽，用于放置布局右侧的内容 |

#### 事件 (Events)
暂无：该组件当前版本未定义特定的交互触发事件。

#### 方法 (Methods)
暂无：该组件当前版本未提供可供外部调用的 API 方法。
