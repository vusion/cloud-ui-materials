# 流程2.0Vue前端组件依赖库（H5端）

## 依赖库介绍
`lcap_process_components_vue_h5` 是专门为 H5 移动端环境设计的流程 2.0 前端组件库。该库旨在为开发者提供一套标准化的流程业务组件，涵盖了从流程发起、审批处理、信息展示到流程追踪的全链路功能。通过高度封装的组件，开发者可以快速在移动应用中集成复杂的流程审批逻辑，确保交互体验的一致性与开发的高效性。

## 包含的组件列表
- **ow-process-button**：流程按钮组件
- **ow-process-form**：流程表单组件
- **ow-process-info**：流程信息组件
- **ow-process-my-process**：我的流程组件
- **ow-process-record**：流程记录组件

## 组件功能说明
1. **ow-process-button（流程按钮）**：用于在流程流转过程中展示各种操作动作（如：提交、同意、拒绝、退回等），封装了流程引擎的动作触发逻辑。
2. **ow-process-form（流程表单）**：流程业务数据的展示容器，支持在流程环境下加载并渲染对应的业务表单内容。
3. **ow-process-info（流程信息）**：展示当前流程实例的关键信息，如标题、编号、发起人、当前节点名称等基础上下文数据。
4. **ow-process-my-process（我的流程）**：提供个人视角下的流程管理界面，包括待办任务、已办任务以及个人发起的流程申请列表。
5. **ow-process-record（流程记录）**：以列表或轴线形式详细展示流程的流转历史，包含各节点的处理人、处理时间及审批意见。
