# 地图导航 (lcap_amap_navigate)

## 依赖库介绍
`lcap_amap_navigate` 是一套专为 Codewave 低代码平台设计的地图导航依赖库。该库集成了高德地图的导航能力，开发者可以通过简单的属性配置，在应用中快速实现目的地导航功能，支持网页版导航展示以及唤起移动端高德地图 App 进行实景导航。

## 包含的组件与逻辑列表

### 组件
- **lcap-amap-nav（地图导航）**: 该组件用于触发地图导航功能，支持设置目的地位置坐标、显示名称及打开方式，并可配置是否尝试唤起高德地图原生 App。

### 逻辑
- 暂无：本依赖库目前仅包含 UI 组件。

## API 说明

### lcap-amap-nav (地图导航)

#### 属性 (Attributes)
| 属性名 | 标题 | 类型 | 默认值 | 说明 |
| :--- | :--- | :--- | :--- | :--- |
| canto | 是否开启导航 | boolean | true | 是否开启导航 |
| lalg | 位置坐标 | string | - | 目的地经纬度坐标（如：116.481488,39.990464） |
| label | 导航名称 | string | - | 导航页标题或目的地显示的名称 |
| target | 打开方式 | string | _self | 链接打开方式，可选值：_blank (新标签页), _self (当前窗口), _parent (父级窗口), _top (顶级窗口) |
| callnativet | 是否调起高德地图APP | boolean | true | 是否在支持的环境下自动调起高德地图原生 APP |

#### 事件 (Events)
暂无。该组件目前未定义任何事件。

#### 方法 (Methods)
暂无。该组件目前未定义任何方法。
