# 地图选点 (lcap_amap_point)

## 依赖库介绍

lcap_amap_point 依赖库专为 Codewave 低代码平台设计，提供与「地图选点」相关的组件。该依赖库主要用于在应用中集成地图交互功能，允许用户通过地图界面搜索、浏览并精确选择地理位置点（POI），从而获取对应的位置名称、详细地址及经纬度坐标信息。

## 包含的组件与逻辑

### 组件
- **lcap-amap-poi (地图选点)**：提供可视化地图界面，支持城市定位、地点搜索及位置选中功能。

### 逻辑
- 暂无内置全局逻辑。

## 组件功能说明

### lcap-amap-poi (地图选点)
用户可以在地图组件中通过手动点击或搜索框查找特定地点。当用户选中某个兴趣点（POI）时，组件会触发相应的事件，并允许通过方法调用的方式导出该点的详细地理信息（如经纬度、格式化地址等），常用于收货地址选择、签到定位等场景。

## API 说明

### lcap-amap-poi

#### 属性 (Attributes)
| 名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| city | string | 默认城市，用于初始化地图显示的中心城市。 |

#### 事件 (Events)
| 名称 | 说明 |
| :--- | :--- |
| select | 点被选中后触发。可用于关联逻辑，例如在选择地点后自动填充表单中的详细地址输入框。 |

#### 方法 (Methods)
| 名称 | 说明 |
| :--- | :--- |
| poiPicked | 获取当前选中 POI 点的具体信息，包括经纬度以及详细地址描述。 |
