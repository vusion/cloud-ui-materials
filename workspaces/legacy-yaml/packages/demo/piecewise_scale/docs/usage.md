# piecewise_scale

## 依赖库介绍
`piecewise_scale` 是一款专为 Codewave 低代码平台打造的依赖库，主要提供「分段缩放」相关的逻辑功能。该库允许开发者定义多个屏幕宽度临界点及其对应的缩放比例，从而在不同分辨率的显示设备上实现更加精细和响应式的界面缩放效果，优化视觉呈现。

## 逻辑列表
- **piecewiseScale**: 分段缩放逻辑，用于根据屏幕宽度计算并应用预设的缩放比例。

## 逻辑功能说明
### piecewiseScale
该逻辑通过配置三个临界宽度（width0, width1, width2）及其对应的缩放倍数（scale0, scale1, scale2），实现在不同分辨率区间内的自动缩放。它默认针对 1920px、2560px 和 3840px 等常见屏幕规格进行了预设，方便开发者快速实现适配。

## API 说明

### 逻辑 piecewiseScale

#### 参数 (Parameters)
| 名称 | 类型 | 默认值 | 说明 |
| :--- | :--- | :--- | :--- |
| width0 | Number | 1920 | 临界宽度0 |
| scale0 | Number | 1 | 临界宽度0对应的缩放比例 |
| width1 | Number | 2560 | 临界宽度1 |
| scale1 | Number | 1.25 | 临界宽度1对应的缩放比例 |
| width2 | Number | 3840 | 临界宽度2 |
| scale2 | Number | 1.75 | 临界宽度2对应的缩放比例 |

#### 属性 (Attributes)
暂无。该项为逻辑组件，不涉及 UI 属性配置。

#### 事件 (Events)
暂无。该逻辑通过显式调用执行，不触发自定义事件。

#### 方法 (Methods)
暂无。该逻辑本身作为一个独立的功能函数调用，内部未定义额外的子方法。
