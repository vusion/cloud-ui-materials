# 昆仑依赖库 (kl_library)

## 依赖库介绍
本依赖库提供与「昆仑依赖库」相关的组件与逻辑，可在 Codewave 低代码平台中引用。主要用于支持树形结构的机构与人员管理展示。

## 包含的组件与逻辑
### 组件
- **kl-tree-view**：树形组件，用于展示机构层级并支持对人员和机构的选择。

### 逻辑
- 暂无相关逻辑。

## API 说明

### kl-tree-view (树形组件)

#### 属性 (Attributes)
| 名称 | 类型 | 默认值 | 说明 |
| :--- | :--- | :--- | :--- |
| value | string | - | 展示最高机构列表 |
| valueUserList | string | - | 选中人员列表（支持双向绑定） |
| valueDeptList | string | - | 选中机构列表（支持双向绑定） |

#### 事件 (Events)
暂无。该组件在当前版本未定义外部事件监听。

#### 方法 (Methods)
暂无。该组件在当前版本未提供外部可调用的方法。
