# cw_drawing_view_library 平面绘制依赖库

## 依赖库介绍
cw_drawing_view_library（平面绘制依赖库）主要用于在 PC 端环境下进行交互式平面绘图。该依赖库支持导入背景底图，并允许用户在背景之上实现各种自定义小部件的拖拽、排布与交互，适用于资产标注、空间规划或简单的绘图应用场景。

## 包含的组件和逻辑列表
### 组件
- **cw-drawing-image-view（平面绘制组件）**：核心绘制视图组件，提供画布管理、背景加载以及小部件的交互控制功能。

### 逻辑
- 暂无逻辑列表

## API 说明

### cw-drawing-image-view (平面绘制组件)

#### 属性 (Attributes)
| 名称 | 类型 | 标题 | 说明 |
| :--- | :--- | :--- | :--- |
| value | string | 组件值 | 组件当前渲染的值内容 |
| comData | Array | 自定义小部件 | 自定义小部件数组格式，例如：{img:"",type:3,title:""} |
| readOnly | Boolean | 只读 | 设置组件是否为只读状态 |

#### 事件 (Events)
| 名称 | 标题 | 说明 | 参数 |
| :--- | :--- | :--- | :--- |
| onSave | 保存后 | 完成保存操作后触发的回调 | - |
| onBindData | 绑定数据 | 进行数据绑定时触发的回调 | - |
| onObjectSelect | 选中对象 | 选中画布中的对象时触发，返回选中的数据列表 | - |

#### 方法 (Methods)
| 名称 | 标题 | 说明 | 参数 |
| :--- | :--- | :--- | :--- |
| handleToImg | 生产图片 | 调用后将当前绘制内容生成图片 | - |
| handleSubmit | 提交数据 | 执行数据提交逻辑 | - |
