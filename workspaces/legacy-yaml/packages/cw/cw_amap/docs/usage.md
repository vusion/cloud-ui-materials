# 高德地图组件 (cw_amap)

## 依赖库介绍
cw_amap 是一款专为 Codewave 低代码平台设计的高德地图功能扩展库。本依赖库旨在简化高德地图能力的接入流程，为开发者提供开箱即用的地图导航和地理信息展示组件。通过集成此库，用户可以快速在低代码应用中实现路线规划、点位标注等核心地理信息服务功能。

## 包含的组件与逻辑列表

### 组件
- **cw-amap-nav**：高德地图导航组件，主要用于实现地图路径规划与实时导航引导。 
- **cw-amap-point-maker-less**：高德地图点标记（精简版）组件，用于在地图视图中快速渲染和展示特定的地理位置标记点。

### 逻辑
- 暂无公开逻辑列表。
