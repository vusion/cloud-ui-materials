# OASDK (cw_oa_sdk)

## 1. 依赖库介绍
本依赖库提供与「OASDK」相关的逻辑功能，旨在为 Codewave 低代码平台提供便捷的 OA 系统对接能力。通过该 SDK，用户可以实现 Token 获取、页面返回控制以及 SDK 初始化等核心操作。

## 2. 包含的逻辑列表
- **getToken**：获取当前环境的 token 信息。
- **goBack**：执行基础的页面返回逻辑。
- **goBackHideFlow**：执行页面返回逻辑，且保持当前流程卡片不被隐藏。
- **init**：执行 SDK 的初始化配置。

## 3. 逻辑 API 说明

### 3.1 getToken
**功能说明**：
获取当前 OA 环境下的 token 信息。

**API 说明**：
- 输入参数：暂无（YAML 中未定义输入参数）
- 输出结果：暂无（YAML 中未定义输出返回值）

### 3.2 goBack
**功能说明**：
调用系统级的 GO_BACK 逻辑，执行页面回退操作。

**API 说明**：
- 输入参数：暂无
- 输出结果：暂无

### 3.3 goBackHideFlow
**功能说明**：
调用页面返回逻辑，特别适用于需要保留当前操作流状态、不隐藏卡片的场景。

**API 说明**：
- 输入参数：暂无
- 输出结果：暂无

### 3.4 init
**功能说明**：
对 OASDK 进行初始化，通常在应用启动或页面挂载时调用。

**API 说明**：
- 输入参数：暂无
- 输出结果：暂无

## 4. 组件说明及 API
暂无组件。该依赖库目前仅包含逻辑（Logic）类型资产。
