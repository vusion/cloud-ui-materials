# cw_pinch_block 缩放容器

## 依赖库介绍
本依赖库提供与「缩放容器」相关的组件与逻辑，可在 Codewave 低代码平台中引用。通过该组件，用户可以实现对容器内容的缩放控制，增强页面的交互展示能力。

## 包含组件列表
- **cw-pinch-block (缩放容器)**: 支持对容器内内容进行缩放操作的布局组件。

## 包含逻辑列表
暂无相关逻辑。
