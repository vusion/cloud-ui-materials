# cw_desensitization 文字脱敏

## 依赖库介绍
cw_desensitization 是一套专为 Codewave 低代码平台定制的文字脱敏逻辑库。该库旨在通过灵活的脱敏策略，帮助开发者在应用开发中对敏感数据（如手机号、姓名、身份证号等）进行遮蔽处理，从而在数据展示和存储环节有效保障隐私安全。

## 包含的逻辑列表
- **desensitizationFromLength**：基于首尾长度脱敏
- **desensitizationFromRegex**：基于正则表达式脱敏

## 功能说明
- **desensitizationFromLength**：该逻辑允许开发者指定字符串开头和结尾需要保留的字符长度，将其余部分替换为脱敏占位符。适用于固定格式或长度已知的脱敏场景。
- **desensitizationFromRegex**：该逻辑支持通过正则表达式匹配敏感信息模式，并根据匹配结果进行精准脱敏替换。适用于匹配复杂规律的脱敏需求。

## API 说明

### 逻辑 desensitizationFromLength
- **属性 (attrs)**：暂无。该逻辑作为运行时功能函数，不涉及可视化组件的属性配置。
- **事件 (events)**：暂无。该逻辑执行后会同步返回处理结果，不产生特定的生命周期事件监听。
- **方法 (methods)**：暂无。该逻辑本身即为一个独立的可执行逻辑单元，未定义额外的内部调用方法。

### 逻辑 desensitizationFromRegex
- **属性 (attrs)**：暂无。此逻辑属于处理型逻辑，不包含前端界面配置相关的属性。
- **事件 (events)**：暂无。逻辑调用为直接触发模式，不涉及交互式的事件反馈机制。
- **方法 (methods)**：暂无。该逻辑被定义为原子执行单元，不包含子层级的方法调用。
