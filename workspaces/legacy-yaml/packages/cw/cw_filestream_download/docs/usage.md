# cw_filestream_download 文件流下载

## 依赖库介绍
本依赖库（cw_filestream_download）专为 Codewave 低代码平台设计，提供核心的「文件流下载」逻辑。其主要功能是允许开发者通过指定的 URL 触发文件流下载，适用于需要从后端接口或外部链接直接下载文件到本地的场景。

## 逻辑列表
- **filestreamDownloadFromUrl**：基于url的文件流下载。该逻辑能够处理给定的 URL 地址，将其指向的文件内容以流的形式下载至用户终端。

## API 说明

### 逻辑 filestreamDownloadFromUrl
**功能说明**：执行基于 URL 的文件流下载逻辑。
