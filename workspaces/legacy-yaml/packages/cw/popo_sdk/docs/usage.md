# popo_sdk 依赖库使用文档

## 依赖库介绍

本依赖库提供与「泡泡依赖库」相关的组件与逻辑，可在 Codewave 低代码平台中引用。

## 包含的组件和逻辑列表

### 逻辑

- **chooseContactsAndSendMessage**：请在这里添加描述
- **chooseContactsAndSendUrl**：唤起选人组件，并给指定的联系人发送网页链接
- **downloadFile**：下载文件到本地
- **getPlatformInfo**：获取系统信息
- **login**：调用login获取登录凭证code
- **navigateBack**：关闭当前页面，返回上一页面
- **share**：分享到POPO聊天会话
- **shareAppToFriend**：分享小程序给好友
- **startWebView**：打开原生WebView
