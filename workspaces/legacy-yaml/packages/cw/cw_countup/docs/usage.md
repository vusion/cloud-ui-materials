# cw_countup 依赖库使用文档

## 依赖库介绍

`cw_countup` 是一个数字动画效果依赖库，主要用于在界面上实现数字从起始值到结束值的平滑渐变动画效果。该库提供了可配置的动画参数，支持循环播放和数字格式化显示，适用于展示统计数据、计数器等需要动态数字变化的场景。

## 包含的组件和逻辑列表

该依赖库包含以下组件：

- **cx-countup**: 数字渐变组件

## 组件功能说明

### cx-countup 组件

数字渐变组件，用于在界面上显示一个从指定起始值平滑过渡到结束值的数字动画。支持配置动画持续时间、循环播放、延迟间隔以及数字格式化选项。

## API 说明

### cx-countup 组件属性

| 属性名 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| start | number | 0 | 渐变开始的数值 |
| end | number | - | 渐变结束的数值 |
| duration | number | 2 | 渐变动画的时间间隔，单位为秒 |
| loop | boolean | false | 是否循环播放动画 |
| delay | number | 1 | 循环播放之间的间隔时间，单位为秒 |
| isGroup | boolean | false | 是否启用数字分隔符（如千位分隔符） |

## 使用说明

该依赖库中的 `cx-countup` 组件可以直接在 Codewave 低代码平台中使用，通过配置不同的属性值来实现不同的数字动画效果。组件会根据配置的起始值、结束值和持续时间自动计算并执行数字渐变动画。

当启用循环播放时，动画会在每次播放完成后等待指定的延迟时间，然后重新开始播放。启用数字分隔符功能后，显示的数字会按照千位分隔符的格式进行显示，提升数字的可读性。

该组件适用于需要动态展示数字变化的场景，如实时数据统计、计数器、进度指示器等。
