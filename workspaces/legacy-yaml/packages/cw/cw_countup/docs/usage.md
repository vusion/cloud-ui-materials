# 依赖库介绍

cw_countup 是一套专为 Codewave 低代码平台设计的「数字渐变」增强库。该库主要通过动态数值滚动的方式，提升数据展示的视觉交互体验，广泛适用于数据看板、统计分析及仪表盘等需要展现数值变化的业务场景。

# 包含的组件与逻辑列表

## 组件列表
- **cx-countup (数字渐变)**：用于实现数字从一个数值平滑过渡到另一个数值的动态滚动效果。

## 逻辑列表
- 暂无相关逻辑。

# API 说明

## cx-countup (数字渐变)

### 属性 (Attrs)

| 名称 | 标题 | 类型 | 默认值 | 说明 |
| :--- | :--- | :--- | :--- | :--- |
| start | 渐变开始数值 | number | 0 | 渐变开始数值 |
| end | 渐变结束数值 | number | - | 渐变结束数值 |
| duration | 渐变时间间隔 | number | 2 | 渐变时间间隔，时间单位为秒 |
| loop | 是否循环播放 | boolean | false | 是否循环播放 |
| delay | 循环播放之间的间隔 | number | 1 | 循环播放之间的间隔，时间单位为秒 |
| isGroup | 是否有分隔符 | boolean | false | 是否有分隔符（如千分位分隔符） |

### 事件 (Events)

暂无。本组件暂未定义可供外部监听的事件。

### 方法 (Methods)

暂无。本组件暂未提供可供外部调用的实例方法。
