# 依赖库介绍

`cw_verifycode_library` 是一个专门为 Codewave 低代码平台设计的依赖库，旨在提供「验证码」相关的组件与业务逻辑。通过集成该库，开发者可以快速在应用中实现图形验证码的展示、刷新及校验功能，增强系统的安全性。

# 包含的组件与逻辑列表

- **组件**：
    - `cw-verifycode-view`（验证码）：用于在页面上渲染图形验证码，支持不同类型的字符组合，并提供刷新和验证的交互能力。
- **逻辑**：
    - 暂无内置逻辑。

# 组件功能说明

### cw-verifycode-view（验证码）
该组件主要负责生成并展示图形验证码。它支持数字、字母以及混合模式，用户可以通过调用组件方法手动刷新验证码内容或触发验证逻辑，同时支持通过属性绑定验证码的当前值。

# API 说明

## cw-verifycode-view

### 属性 (Attributes)
| 名称 | 类型 | 默认值 | 说明 |
| :--- | :--- | :--- | :--- |
| value | string | - | 需要传入的值（支持双向绑定） |
| type | string | 'number' | 验证码类型。可选值：blend (混合)、number (数字)、letter (字母) |

### 事件 (Events)
暂无自定义事件。

### 方法 (Methods)
| 名称 | 标题 | 说明 |
| :--- | :--- | :--- |
| validate | 验证 | 执行验证码校验操作 |
| refresh | 刷新 | 重新生成并刷新验证码内容 |
