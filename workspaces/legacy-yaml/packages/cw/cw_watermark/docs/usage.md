# cw_watermark (水印)

## 依赖库介绍
cw_watermark 是一个专为 Codewave 低代码平台设计的组件库，旨在提供灵活且易于配置的全局水印功能。通过该库，开发者可以轻松地在应用中添加图片或文字形式的水印，从而增强应用的版权保护、信息安全及品牌识别度。

## 包含的组件与逻辑

### 组件
- **watermark (全局图片水印)**：用于在页面上生成覆盖式的图片或文字水印，支持自定义样式与间距。

### 逻辑
- 暂无

## 组件功能说明

### watermark (全局图片水印)
该组件支持两种模式：图片水印和文字水印。它能够根据配置参数自动平铺水印内容。开发者可以自由调整文字的大小、颜色、间距，或者指定图片的 URL 地址及透明度，以达到最佳的视觉展示效果。

## API 说明

### watermark (全局图片水印)

#### 属性 (Attributes)
| 属性名称 | 类型 | 默认值 | 说明 |
| :--- | :--- | :--- | :--- |
| type | string | image | 水印类型。可选值：image (图片水印)、text (文字水印) |
| text | string | "" | 文字水印的内容，仅在 type 为 text 时生效 |
| color | string | "#000" | 文字水印的颜色，支持 CSS 颜色格式，仅在 type 为 text 时生效 |
| fontSize | number | 16 | 文字水印的字体大小，仅在 type 为 text 时生效 |
| cSpace | number | 20 | 文字水印的水平间距，仅在 type 为 text 时生效 |
| vSpace | number | 50 | 文字水印的垂直间距，仅在 type 为 text 时生效 |
| src | image | "" | 水印图片的 URL 地址，仅在 type 为 image 时生效 |
| opacity | number | - | 水印内容的透明度 |

#### 事件 (Events)
- 暂无：该组件目前未定义任何外部触发事件。

#### 方法 (Methods)
- 暂无：该组件目前未提供可被外部调用的逻辑方法。
