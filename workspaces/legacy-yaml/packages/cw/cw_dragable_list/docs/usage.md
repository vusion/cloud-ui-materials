# cw_dragable_list 可拖拽组件列表

## 依赖库介绍
cw_dragable_list 依赖库旨在为 Codewave 低代码平台提供灵活的「可拖拽组件列表」功能。通过该组件库，开发者可以快速实现列表项的拖拽排序以及不同列表间的项移动，适用于任务看板、自定义排序列表等业务场景。

## 包含的组件和逻辑列表
- **cw-dragable-list（可拖拽数据列表）**：核心组件，用于展示并管理可拖拽的数据项。

## 组件和逻辑功能说明
- **cw-dragable-list**：该组件支持绑定动态数据源，并允许用户通过鼠标或触摸操作调整列表项的顺序。它支持设置分组以实现跨列表拖拽，并提供横向和纵向两种布局方向。

## API 说明

### cw-dragable-list (可拖拽数据列表)

#### 属性 (Attrs)
| 属性名 | 标题 | 类型 | 默认值 | 说明 |
| :--- | :--- | :--- | :--- | :--- |
| data-source | 数据源 | array | - | 展示数据的输入源，可设置为数据集对象或者返回数据集的逻辑。 |
| data-schema | 数据类型 | schema | - | IDE 根据配置的数据源动态计算返回内容的数据结构，用于说明 current.item 的类型。 |
| group | 分组 | string | group | 相同分组名称的组件之间可以互相拖拽元素。 |
| direction | 排列方向 | string | horizontal | 列表的方向。可选值：horizontal (横向), vertical (纵向)。 |

#### 事件 (Events)
暂无。该组件目前未定义特定的事件响应。

#### 方法 (Methods)
暂无。该组件目前未提供外部调用的 API 方法。

#### 插槽 (Slots)
- **default**: 内容自定义插槽。提供插槽属性 `current`，其中包含当前项的数据（T）。
