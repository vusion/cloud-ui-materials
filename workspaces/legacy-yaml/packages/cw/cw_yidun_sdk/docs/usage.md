# 易盾sdk (cw_yidun_sdk)

## 依赖库介绍

`cw_yidun_sdk` 是一套专为 Codewave 低代码平台设计的集成工具库。本依赖库主要集成了「易盾（NetEase Yidun）」验证码能力，旨在为应用提供安全、便捷的人机验证逻辑，有效防止恶意刷票、暴力破解等风险行为。

## 包含的组件与逻辑列表

### 组件
- **cw-yidun-view（易盾）**: 用于在页面中嵌入易盾验证码交互界面，支持配置基础 ID 信息，并处理验证过程中的成功、失败等状态反馈。

### 逻辑
- 暂无逻辑列表内容。

## API 说明

### cw-yidun-view (易盾)

#### 属性 (Attributes)
| 名称 | 类型 | 说明 |
| :--- | :--- | :--- |
| containerId | string | 易盾组件id,用于区分多个地方使用 |
| captchaId | string | 易盾captchaId,由易盾官方提供 |

#### 事件 (Events)
| 名称 | 标题 | 参数 | 说明 |
| :--- | :--- | :--- | :--- |
| onVerify | 验证成功事件 | value: string | 验证成功时触发，传出验证相关的值 |
| onVerifyError | 验证失败事件 | value: string | 验证失败时触发，传出错误提示或状态 |

#### 方法 (Methods)
| 名称 | 标题 | 参数 | 说明 |
| :--- | :--- | :--- | :--- |
| verify | 验证 | 无 | 手动触发验证流程 |
| refresh | 重置 | 无 | 重置验证码状态，使其恢复到初始态 |
