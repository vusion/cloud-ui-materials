# 自由移动布局 (cw_active_room_library)

本依赖库提供了一套高度灵活的布局管理方案，允许在编辑台中自定义切换布局的顺序以及每一栏的具体宽度。通过容器与单元组件的配合，实现可拖拽、可调节的动态页面布局结构。

## 包含组件

- **cw-active-drag-view（活动布局容器）**：作为布局的主容器，负责处理内部单元的拖拽逻辑、数据源管理及状态切换。
- **cw-active-drag-view-cell（可拖拽单元）**：布局的基本单元，需放置于容器内部，支持通过属性控制自身所占的宽度百分比。

## API 说明

### cw-active-drag-view (活动布局容器)

#### 属性 (Attributes)
| 属性名 | 说明 | 类型 | 默认值 |
| :--- | :--- | :--- | :--- |
| value | 需要传入的数据源 | Array | - |

#### 事件 (Events)
| 事件名 | 说明 | 参数 |
| :--- | :--- | :--- |
| onSaveData | 保存数据时触发 | - |

#### 方法 (Methods)
| 方法名 | 说明 | 参数 |
| :--- | :--- | :--- |
| handleChangeActive | 切换为活动视图 | - |
| handleSaveData | 保存数据 | - |

### cw-active-drag-view-cell (可拖拽单元)

#### 属性 (Attributes)
| 属性名 | 说明 | 类型 | 默认值 |
| :--- | :--- | :--- | :--- |
| value | 单元宽度（请使用百分比） | string | 25% |

#### 事件 (Events)
该组件暂无监听事件。

#### 方法 (Methods)
该组件暂无对外公开的调用方法。
