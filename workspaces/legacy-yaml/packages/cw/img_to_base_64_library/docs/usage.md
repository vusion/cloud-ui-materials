# img_to_base_64_library 依赖库使用文档

## 依赖库介绍

本依赖库提供与「图片url转base64」相关的组件与逻辑，可在 Codewave 低代码平台中引用。

## 包含的组件和逻辑列表

### 逻辑

- **base64ToFile**：base64转化成File
- **contentToFile**：内容转文件链接并下载
- **imgToBase64**：图片url转base64
