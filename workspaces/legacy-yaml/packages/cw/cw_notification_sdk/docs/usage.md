# cw_notification_sdk 全局提示

## 依赖库介绍
cw_notification_sdk 是一个专为 Codewave 低代码平台设计的全局提示依赖库。该库封装了与通知（Notification）相关的核心逻辑，旨在帮助开发者在应用运行过程中，通过简单配置即可实现各种全局提示反馈，提升用户交互体验。

## 包含列表
### 逻辑列表
- **notification**：通知逻辑，用于触发全局提示信息。

## 功能说明
- **notification**：该逻辑主要负责在前端运行时环境下，向用户发起通知。它可以被集成在各种业务流中，用于显示操作反馈、系统警告或状态更新等信息。
