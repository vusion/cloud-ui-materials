# 流程依赖库 (cw_flow_chart_library)

## 依赖库介绍
本依赖库提供与「流程依赖库」相关的组件与逻辑，可在 Codewave 低代码平台中引用。主要旨在为低代码应用提供流程图的绘制、展示及数据导出的功能支持。

## 包含的组件与逻辑列表
- cw-flow-simble-view (简单流程绘制)：提供轻量化的流程图渲染与交互能力。

## API 说明

### cw-flow-simble-view (简单流程绘制)

#### 属性 (Attributes)
| 名称 | 类型 | 标题 | 说明 |
| :--- | :--- | :--- | :--- |
| value | string | 值 | 需要传入的流程图定义或配置字符串 |

#### 事件 (Events)
暂无。该组件暂未定义任何事件。

#### 方法 (Methods)
| 名称 | 标题 | 参数 | 说明 |
| :--- | :--- | :--- | :--- |
| exportPNG | 生成图片 | - | 触发流程图的图片导出功能 |
| toPNG | 生成Base64图片 | - | 生成流程图的 Base64 格式图片数据 |
| toJSON | 生成数据 | - | 获取流程图当前的状态或定义数据 |
