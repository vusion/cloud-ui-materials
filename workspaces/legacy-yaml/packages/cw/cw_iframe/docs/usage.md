# 依赖库介绍

`cw_iframe` 是一套专为 Codewave 低代码平台设计的扩展库，主要提供 iframe 嵌入功能。通过本库提供的组件，开发者可以在低代码应用中方便地集成第三方网页、外部链接或其他 Web 资源，实现跨域内容展示与交互整合。

# 包含组件与逻辑

## 组件
- **iframe (cw-iframe)**: 核心嵌入组件，用于在页面指定区域内渲染外部网页内容。

（暂无相关逻辑）

# API 说明

## iframe (cw-iframe)

### 属性 (attrs)

| 名称 | 标题 | 类型 | 默认值 | 说明 |
| :--- | :--- | :--- | :--- | :--- |
| src | 地址 | string | - | 需要嵌入的网页地址 |

### 事件 (events)

| 名称 | 标题 | 参数 | 说明 |
| :--- | :--- | :--- | :--- |
