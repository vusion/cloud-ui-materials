# 飞书sdk (cw_feishu_sdk)

本依赖库提供与「飞书sdk」相关的组件与逻辑，旨在简化开发者在 Codewave 低代码平台中接入飞书能力的过程。通过封装飞书核心功能，支持应用鉴权、免等授权及相关组件的快速集成。

## 包含列表

### 组件
- **cw-feishu-demo**: 飞书接入组件测试，用于验证飞书环境下的基础交互。

### 逻辑
- **chooseContact**: 调用飞书原生能力，实现联系人选择功能。
- **config**: 应用鉴权逻辑，用于初始化飞书 SDK 并校验权限。
- **requestAuthCode**: 免等授权逻辑，用于获取临时授权码以实现免登流程。

## API 说明

### 组件: cw-feishu-demo (飞书接入组件测试)

#### 属性 (attrs)
| 名称 | 类型 | 默认值 | 说明 |
| :--- | :--- | :--- | :--- |
| value | string | 请在这里编写代码 | 需要传入的值 |

#### 事件 (events)
暂无（该组件目前未定义任何交互事件）

#### 方法 (methods)
暂无（该组件目前未定义任何可调用的实例方法）

### 逻辑: chooseContact (选择联系人)
该逻辑用于在飞书客户端内唤起联系人选择界面，方便用户挑选目标人员。

### 逻辑: config (应用鉴权)
该逻辑用于完成飞书 JS-SDK 的鉴权配置，是调用其他飞书原生能力的前提条件。

### 逻辑: requestAuthCode (免等授权)
该逻辑用于获取当前登录用户的免登授权码，通常用于后续换取用户信息。
