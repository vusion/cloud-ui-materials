# 应用模板预览组件 (cust_preview_app_template)

## 依赖库介绍
本依赖库提供与「应用模板预览组件」相关的组件与逻辑，专门为 Codewave 低代码平台设计。其核心功能是提供应用模板的预览能力，方便开发者在低代码应用中集成模板展示与交互逻辑。

## 包含的组件和逻辑列表
- **cust_preview (预览)**: 用于实现应用模板的预览展示组件。
