# 前端数据权限注解依赖库 (lcap-fe-annotation-data-permission)

## 依赖库介绍
lcap-fe-annotation-data-permission 是一款专为 Codewave 低代码平台设计的前端数据权限注解依赖库。该库主要提供前端数据权限相关的注解定义与解析功能，旨在辅助平台在应用运行或编译阶段对数据访问权限进行精细化控制，提升应用的数据安全管理能力。

## 包含的组件和逻辑列表
- 组件：暂无
- 逻辑：暂无

## 每个组件和逻辑的功能说明
暂无。本依赖库目前主要作为底层注解协议库使用，暂未导出可直接在页面中使用的 UI 组件或业务逻辑函数。
