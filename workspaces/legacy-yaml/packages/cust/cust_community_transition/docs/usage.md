# 社区transition库 (cust_community_transition)

## 依赖库介绍
本依赖库提供与「社区transition库」相关的组件与逻辑，可在 Codewave 低代码平台中引用，主要用于为页面内容提供过渡动画容器支持。

## 包含的组件与逻辑
- **cust-community-transition** (社区transition容器): 核心功能组件，用于包裹页面元素并应用指定的过渡动画效果。

## 组件 API 说明

### cust-community-transition (社区transition容器)

#### 属性 (attrs)
| 名称 | 类型 | 默认值 | 说明 |
| :--- | :--- | :--- | :--- |
| name | string | name | 过渡效果的名称 |

#### 事件 (events)
暂无。该组件目前未定义任何交互事件。

#### 方法 (methods)
暂无。该组件目前未提供可调用的外部方法。

#### 插槽 (slots)
- **default**: 用于放置需要应用过渡效果的内容。
