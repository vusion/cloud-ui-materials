# cust_community_icon_image 社区图标图片

本依赖库提供与「社区图标图片」相关的组件与逻辑，可在 Codewave 低代码平台中引用。该库旨在帮助开发者快速集成和展示社区风格的图标资源，提供灵活的样式配置能力。

## 包含组件

- **cust-community-icon-image (社区图标图片)**：专门用于展示社区图标的组件，支持通过 URL 或本地路径指定图片源，并允许自定义复杂的样式属性。
