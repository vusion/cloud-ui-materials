# 省烟草sdk (cust_tobacco_sdk)

## 依赖库介绍
本依赖库提供与「省烟草sdk」相关的核心逻辑功能，封装了业务系统中常用的埋点监控、用户信息获取、地理定位以及权限校验等能力。通过集成该 SDK，开发者可以在 Codewave 低代码平台中便捷地调用省烟草业务接口，提升开发效率并确保业务逻辑的一致性。

## 逻辑列表
- **addBizAccessLog**：记录业务埋点，用于追踪业务流程节点。
- **addFunAccessLog**：记录功能埋点，用于统计功能模块的访问频率。
- **getAccountInfo**：获取当前登录用户的详细账户信息。
- **locate**：调用设备的 GPS 模块获取地理位置经纬度。
- **mockAnywareObject**：模拟 anyware 对象，以便在测试环境中进行逻辑验证。
- **ready**：环境准备状态检查，确保 SDK 资源已加载完成。
- **requestAuthCode**：发起授权请求，获取用于业务鉴权的用户 Token。
