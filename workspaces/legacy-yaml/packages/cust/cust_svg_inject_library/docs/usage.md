# SVG图片依赖库 (cust_svg_inject_library)

本依赖库提供与「SVG图片依赖库」相关的组件与逻辑，可在 Codewave 低代码平台中引用。主要功能是支持 SVG 图片的注入与视图展示。

## 包含的组件与逻辑列表

### 组件
- **cust-svg-inject-view（SVG视图）**: 用于在页面中渲染和展示指定的 SVG 图片内容。

### 逻辑
- 暂无逻辑列表。

## API 说明

### cust-svg-inject-view (SVG视图)

#### 属性 (attrs)
| 名称 | 类型 | 默认值 | 说明 |
| :--- | :--- | :--- | :--- |
| value | image | - | 图片地址，需要传入的值 |

#### 事件 (events)
暂无自定义事件说明。

#### 方法 (methods)
暂无自定义方法说明。
