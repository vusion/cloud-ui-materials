# 批量下载 (cust_multi_download)

## 依赖库介绍
`cust_multi_download` 依赖库专为 Codewave 低代码平台设计，主要提供批量下载相关的业务逻辑封装。通过集成该库，开发者可以实现针对上传组件中多个文件的批量导出与统一下载，提升应用的文件处理效率。

## 包含的组件和逻辑列表
- **组件**：无
- **逻辑**：uploadMultiDownload

## 功能说明
- **uploadMultiDownload**：该逻辑用于处理上传组件产生的多个文件，执行批量下载操作，简化文件管理流程。

## API 说明
